## 前言

今天给大家介绍一个无敌好用的软件，它就是 **Raycast**。每天都会用到它，日常工作基本离不开了，有很多便捷的插件，可以大大提高工作效率。

## 安装 RayCast

RayCast 官网：[https://www.raycast.com](https://www.raycast.com)

```bash
brew install --cask raycast
```

`Command + Space` 启动 RayCast
